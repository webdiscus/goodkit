  • .h-{sm|md|lg|xl}-{25|50|75|100|125|150|auto} - controls the height of an element in %.
  • .w-{sm|md|lg|xl}-{25|50|75|100|125|150|auto} - controls the width of an element in %.
  • .mw-{sm|md|lg|xl}-{25|50|75|100|125|150|auto} - controls the max width of an element in %.

  • .vw-{sm|md|lg|xl}-{25|50|75|100|125|150|auto} - controls the width of an element in vw.
  • .max-vw-100 - sets the max width of an element to 100vw.
